## 介绍
- 基于laravel5.8+vuejs+adminlte构建的后台管理系统
## 安装
- composer install
- php artisan migrate
- php artisan db:seed
- npm install (linux) | npm install --no-bin-links (windows)
- npm install cross-env (如果npm run dev 无法执行的话)
